# 低GI成品食品

本目录收录市面上可直接购买的低GI成品食品，供社区成员参考和分享。每条记录包含产品信息、GI值估计、营养成分和购买渠道。

> **⚠️ 注意**：本知识库不构成购买推荐。GI值和营养成分以产品包装标注为准，不同批次可能存在差异。

## 如何贡献

如果你想添加新的成品食品，请按以下步骤操作：

1. 确认产品包装上标注了相关营养信息
2. 在产品对应分类目录下创建 `.md` 文件
3. 填写完整的 YAML 元数据（type、title、brand、description 等）
4. 提交 Pull Request

详见 [贡献指南](https://github.com/thomaszta/lowgi-community.github.io/blob/master/CONTRIBUTING.md)。
