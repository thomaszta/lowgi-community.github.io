# 魔芋面 / 蒟蒻面

## 产品信息

- **主要成分**：魔芋精粉、水、食用碱
- **常见形态**：面条状、米粒状、块状
- **保质期**：未开封常温 12 个月（袋装水煮型）；冷藏保存

## GI值估计

约 0-5（极低GI）。魔芋的主要成分是葡甘露聚糖——一种几乎不被人体消化吸收的可溶性膳食纤维。

## 营养成分（每100g）

| 项目 | 含量 |
|------|------|
| 能量 | 约 5-10 kcal |
| 蛋白质 | < 1g |
| 脂肪 | < 0.5g |
| 碳水化合物 | < 3g（其中膳食纤维约 2-3g） |
| 钠 | 因品牌而异（建议低钠选择） |

## 食用方法

1. 开袋沥去水分
2. 用清水冲洗 2-3 次去除碱味
3. 沸水焯烫 1-2 分钟
4. 沥干后加入汤底或酱料中加热

## 选购要点

- 选择**无添加糖**的产品
- 注意钠含量，部分产品含较多盐分
- 袋装水煮型比即食型口感更好

## 食用建议

- 魔芋面本身几乎没有味道，适合搭配浓汤底或酱料
- 搭配蛋白质（鸡肉、豆腐、鸡蛋）和蔬菜，营养更均衡
- 适合代替米粉、面条用于汤粉、凉拌等
