# Low-GI Commercial Products

This section catalogs commercially available low-GI packaged foods for community reference. Each entry includes product information, estimated GI value, nutritional profile, and purchasing tips.

> **⚠️ Disclaimer**: This knowledge base does not constitute product endorsement. GI values and nutritional information should be verified against product labels, as formulations may vary between batches.

## How to Contribute

To add a new commercial product:

1. Verify the nutritional information on the product packaging
2. Create a `.md` file in the appropriate category directory
3. Fill in the complete YAML frontmatter (type, title, brand, description, etc.)
4. Submit a Pull Request

See [Contributing Guidelines](https://github.com/thomaszta/lowgi-community.github.io/blob/master/CONTRIBUTING.md) for details.
